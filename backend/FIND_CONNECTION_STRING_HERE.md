## Exact Steps:

1. **You're in the Database section** - Good!

2. **Look at the "CONFIGURATION" section** (in grey text)

3. **Click on "Settings"** - This is what you need!

   It's located:
   - Under "CONFIGURATION"
   - Below "Roles"
   - Below "Policies"
   - Click on **"Settings"**

4. **On the Settings page**, you should see:
   - Connection string
   - Connection pooling
   - Database URL
   - Or similar connection information

5. **Copy the connection string** that looks like:
   ```
   postgresql://postgres:[PASSWORD]@...
   ```

## What You're Looking For

Once you click "Settings", look for:
- A field labeled "Connection string"
- A field labeled "URI"
- A field labeled "Database URL"
- Something that starts with `postgresql://`

Copy that entire string and we'll use it!
